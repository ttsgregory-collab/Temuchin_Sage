# Drafts & Explorations

This directory is for **work-in-progress**, raw writing, scene fragments, and experimental sections.

**No pressure here.** This is where rough ideas live before they become chapters.

---

## How to Use This Folder

- **Raw scenes:** Write without editing. Let it be messy.
- **Fragments:** Scenes you might use later, even if they're incomplete.
- **Alternatives:** Multiple versions of the same scene or memory.
- **Explorations:** Writing that might not make it into the final memoir.
- **Dialogue tests:** Trying out conversations or voice.

When a draft feels ready to move to `chapters/`, rename it and relocate it.

---

**Commit often.** Every version is history. Nothing is lost.
