# Assets

This directory holds images, documents, and reference materials for the memoir.

---

## Supported Files

- **Images:** Photos, scans of old documents, maps (.jpg, .png, .gif)
- **Documents:** PDFs, historical records, letters (if referenced)
- **Media:** Any supporting material that illustrates or grounds the narrative

---

## Naming Convention

Use descriptive names:
```
assets/childhood-home-1985.jpg
assets/family-photo-summer-1990.jpg
assets/letter-from-mother.pdf
assets/newspaper-clipping-1995.pdf
```

## Usage in Chapters

Reference images in Markdown like this:

```markdown
![Photo of childhood home](../assets/childhood-home-1985.jpg)
```

---

**Note:** Keep file sizes reasonable. Large media can slow down the repository.
