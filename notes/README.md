# Notes & Research

This directory holds supporting materials for the memoir.

---

## Contents

- **Character sketches:** Descriptions, backgrounds, relationships
- **Timeline:** Historical context, dates, eras
- **Locations:** Detailed descriptions of key places
- **Research:** Fact-checking, references, historical context
- **Themes:** Explorations of central ideas or motifs
- **Reflections:** Present-day thoughts on past events

---

## Organization

Organize by type or theme:
```
notes/characters.md
notes/timeline.md
notes/locations.md
notes/historical-context.md
notes/theme-loss.md
notes/reflections.md
```

These are **reference materials** — they don't appear in the final memoir, but they help you write more clearly and authentically.
